## 실행방법

### 1. 의존성 설치

```bash
uv sync
```

### 2. 서버 실행

#### 방법 1: Python 스크립트로 실행
```bash
uv run python main.py
```

#### 방법 2: uvicorn으로 직접 실행
```bash
uv run uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### 3. API 접근

서버 실행 후 다음 URL로 접근할 수 있습니다:

| URL | 설명 |
|-----|------|
| http://localhost:8000/ | 기본 API 엔드포인트 |
| http://localhost:8000/redoc | ReDoc API 문서 |

### 사용자 관리

#### 시나리오 1: 회원가입
- Given: 새로운 사용자 정보가 주어졌을 때
- When: 회원가입 API를 호출하면
- Then: 사용자가 정상적으로 생성된다

#### 시나리오 2: 중복 이메일 회원가입
- Given: 이미 존재하는 이메일로
- When: 회원가입을 시도하면
- Then: 409 Conflict 에러가 발생한다

#### 시나리오 3: 로그인
- Given: 등록된 사용자 정보가 있을 때
- When: 올바른 이메일과 비밀번호로 로그인하면
- Then: 로그인이 성공한다

#### 시나리오 4: 로그아웃
- Given: 로그인된 사용자가 있을 때
- When: 로그아웃 API를 호출하면
- Then: 로그아웃이 성공한다

#### 시나리오 5: 회원정보 조회
- Given: 등록된 사용자가 있을 때
- When: 회원정보 조회 API를 호출하면
- Then: 사용자 정보가 반환된다

#### 시나리오 6: 회원정보 수정
- Given: 등록된 사용자가 있을 때
- When: 회원정보 수정 API를 호출하면
- Then: 사용자 정보가 업데이트된다

#### 시나리오 7: 회원 탈퇴
- Given: 등록된 사용자가 있을 때
- When: 탈퇴 API를 호출하면
- Then: 사용자가 삭제된다
